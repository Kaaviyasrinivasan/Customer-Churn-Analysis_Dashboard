📊 Customer Churn Analysis Dashboard – Excel
image
A professional and interactive Customer Churn Analysis Dashboard built in Microsoft Excel to analyze customer retention, churn behavior, and key risk factors using dynamic visual analytics.

This dashboard helps businesses identify why customers churn, recognize high-risk segments, and design data-driven retention strategies.

🚀 Features
🎛️ Interactive Filters
Analyze churn data dynamically using Excel slicers:

Payment Method: Bank Transfer, Credit Card, Electronic Check, Mailed Check
Internet Service: DSL, Fiber Optic, No Internet
Contract Type: Month-to-Month, One Year, Two Year
Tenure: < 1 Year, 1–2 Years, 2+ Years
Gender: Male, Female
Senior Status: Senior, Non-Senior

📈 Visual Analytics
Churn by Internet Service Type
Fiber Optic customers show the highest churn rate compared to DSL and No Internet users.

Churn by Customer Tenure
Customers with less than one year of tenure have the highest churn rate, highlighting early retention risk.

Churn by Gender
Churn distribution is almost equal between male and female customers.

Churn by Senior Status
Non-senior customers contribute the majority of churn, while senior customers represent a smaller but notable segment.

Impact of Contract Type on Churn
Month-to-month contracts experience extremely high churn compared to long-term contracts.

Churn by Payment Method
Customers using electronic check have the highest churn rate among all payment methods.
